How-to, this model based on evolution-strategy

  1. You can check realtime-evolution-strategy.ipynb for to train an evolution strategy to do realtime trading.

I trained the model to learn trading on different stocks,

python ['TWTR.csv', 'GOOG.csv', 'FB.csv', 'LB.csv', 'MTDR.csv', 'CPRT.csv', 'FSV.csv', 'TSLA.csv', 'SINA.csv', 'GWR.csv']

You might want to add more to cover more stochastic patterns.

  1. Run app.py to serve the checkpoint model using Flask,

bash python3 app.py
