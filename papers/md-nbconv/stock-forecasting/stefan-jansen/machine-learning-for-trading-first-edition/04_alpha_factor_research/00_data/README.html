Engineering alpha factors

Based on a conceptual understanding of key factor categories, their rationale and popular metrics, a key task is to identify new factors that may better capture the risks embodied by the return drivers laid out previously, or to find new ones.

Useful pandas and NumPy methods

NumPy and pandas are the key tools for custom factor computations. The Notebook feature_engineering contains examples of how to create various factors.

The notebook uses data generated by the notebook create_datasets script in the data folder in the root directory of this GitHub repo and stored in HDF5 format for faster access.

See the notebook storage_benchmarks in the directory for Chapter 2, Market and Fundamental Data for a comparison of parquet, HDF5, and csv storage formats for pandas DataFrames.
